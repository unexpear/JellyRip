### What's New in 1.0.10

Tool discovery and validation
- Unified resolver for MakeMKV and ffprobe: saved config -> common paths -> PATH.
- validate_makemkvcon and validate_ffprobe run a real probe command before accepting a path.
- Settings save refuses to overwrite a working tool path with an unvalidated replacement.
- Engine validate_tools routes both dependencies through the resolver layer.

Multi-disc and extras workflow
- Multi-disc dump mode renamed from Unattended to Dump All.
- Between-disc confirmation prompt (no default timeout).
- Unrecognized discs can be bypassed or stopped instead of retry-only.
- Extras selection changed from yes/no to multi-select picker.
- Configurable prompt auto-timeouts and disc-swap timeouts in Advanced settings.

Settings crash hardening
- Settings open callback wrapped in safety handler; exceptions show error dialog not crash.

Project and documentation
- Added CONTRIBUTING.md, SECURITY.md, pyproject.toml, GitHub Actions CI workflow.
- Added docs/architecture.md and docs/repository-layout.md.
- Version strings synchronized across all files.
- Fixed Unicode mojibake in controller.
