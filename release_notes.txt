### What's New in 1.0.12

Disc-level metadata (CINFO parsing)
- Scan now extracts disc title, language, and volume ID from MakeMKV CINFO output.
- Fallback naming prefers CINFO disc name over per-title TINFO when available.

MakeMKV --minlength filter
- New setting: minimum title length in seconds (Advanced > MakeMKV, default off).
- Titles shorter than the threshold are excluded during scan.

Jellyfin metadata ID support
- Smart Rip, Manual Disc, and Organize flows prompt for optional TMDB/IMDB/TVDB ID.
- Folder names get Jellyfin-compatible tags: Movie (2024) [tmdbid-12345].
- Accepts flexible input: tmdb:12345, tt1234567, tvdb:79168, or bare numbers.

Build and encoding fixes
- Fixed 73 corrupted em dashes in controller (mojibake from triple-encoded UTF-8).
- 34 new naming tests (180 total).
- Fixed Unicode mojibake in controller.
