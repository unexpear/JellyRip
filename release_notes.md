## What's New

- Added per-title Preview support directly in the title picker.
- Preview now runs as a short disposable sample rip and can auto-open in VLC.
- Added workflow-level resume with restored metadata context (title, year, season, selected titles, episode fields).
- Added input prefill so interrupted sessions can resume with less manual re-entry.

## Reliability Improvements

- Enforced all-or-nothing rip result handling.
- Invalid, missing, aborted, or failed-title outputs are treated as session failure.
- Failed sessions wipe media outputs while preserving metadata for workflow resume.

## UI and Settings

- Added naming mode options: timestamp, auto title, and auto title + timestamp.
- Reorganized Settings layout into clearer grouped sections.
